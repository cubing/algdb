# AlgDB

AlgDB is an algorithm database for Twisty Puzzles currently in development.

## Getting Started

### Client

Read about the client structure in the [wiki](https://github.com/cubing/algdb/wiki/Client-side-route-structure)

```shell
cd client
yarn
yarn start
```
